🛠️ Setup

Prerequisites & Setup

Get your environment ready for building AI code reviewers. This should take about 15 minutes.

15 min

Required Tools

Claude Code

You need Claude Code with an active subscription (Pro, Teams, or Enterprise). Verify installation:

bash 
# Verify Claude Code is installed
claude --version

# Create the agents directory
cd your-project
mkdir -p .claude/agents
echo "Setup complete!"

Git and Command Line

You should be comfortable with:

  • Basic Git commands (git add, git commit, git diff)
  • Navigating directories in a terminal
  • Running shell commands

A Code Project to Practice On

Have a real project with at least a few hundred lines of code. The subagents you build will review this code, so pick something you know well.

💡
Use a Project You Know

Testing code reviewers on familiar code helps you evaluate whether they're catching real issues or generating noise.

Recommended Setup

  • VS Code with Claude Code extension for inline suggestions
  • GitHub account for testing with real repos and PRs
  • Basic understanding of code review principles

Course Materials Location

All agent files will be created in .claude/agents/ within your project:

text 
your-project/
├── .claude/
│   └── agents/
│       ├── bug-finder.md
│       ├── code-reviewer.md
│       ├── security-scanner.md
│       └── test-generator.md
├── src/
└── tests/
