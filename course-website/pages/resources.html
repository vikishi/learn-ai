Agent Template

Use this skeleton to create new agents:

markdown
---
name: your-agent-name
description: Specific description of what this agent does (Claude uses this to decide when to delegate)
tools: Read, Bash, Grep, Glob
---

# Agent Name

You are a specialist focused on [specific domain].

## Mission
[What this agent does and why]

## Process
1. [First step]
2. [Analysis steps]
3. [Output results]

## What to Look For
- [Category 1]
- [Category 2]
- [Category 3]

## Output Format
[Exact template for results]

## What NOT to Report
- [Things to skip]
- [False positive avoidance]

## Project Context (optional)
[Add project-specific rules here]

Troubleshooting

IssueSolution
Agent not triggeringCheck file path (.claude/agents/[name].md), verify YAML frontmatter, make description specific, try explicit: "Use the [name] agent"
Too many false positivesAdd filtering rules, increase severity threshold, add verification steps
Missing real issuesAdd specific checks, improve examples, lower detection threshold
Inconsistent outputAdd strict template, use exact format specification, include example output
Agent too slowReduce scope, use faster tools (Grep over Bash), limit file count

Post-Course Challenges

Week 1: Daily Usage

  • ☐ Review every file before commit
  • ☐ Track bugs caught and time saved
  • ☐ Note frustrations and false positives
  • ☐ End of week: review log and refine

Week 2: Refinement

  • ☐ Update instructions based on Week 1 false positives
  • ☐ Add project-specific rules
  • ☐ Create 1-2 workflow shortcuts
  • ☐ Measure improvement vs. Week 1

Week 3: Build Custom Agent

Choose based on your biggest pain point:

  • Slow reviews → Better reviewer with context
  • Frequent bugs in area X → Specialized scanner
  • Low test coverage → Enhanced test generator
  • Code quality issues → Custom analyzer

FAQ

Can I use these in any project?

Yes! Copy the .claude/agents/ directory into any project root.

How do I share with my team?

Commit .claude/agents/ to git. Every team member with Claude Code will automatically see the agents.

Do subagents work with all languages?

Yes, but you may want language-specific versions for deeper analysis.

How often should I update them?

Review monthly. Update when you notice patterns in false positives or missed issues.
